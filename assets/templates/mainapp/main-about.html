About The Project

This project aims to use machine learning algorithms to predict the modes of childbirth for expectant mothers. By analyzing a variety of factors, including medical history and prenatal health, the model will provide personalized predictions to improve birth outcomes and support informed decision-making for mothers and healthcare providers.

Algorithms Used

We have used 3 algorithms in this project to make a comparison to find the most accurate prediction.

Logistic Regression

This type of statistical model (also known as logit model) is often used for classification and predictive analytics. Logistic regression estimates the probability of an event occurring, such as voted or didn't vote, based on a given dataset of independent variables.

Gradient Boost

Gradient boosting is a type of machine learning boosting. It relies on the intuition that the best possible next model, when combined with previous models, minimizes the overall prediction error. The key idea is to set the target outcomes for this next model in order to minimize the error.

XG Boost

XGBoost, which stands for Extreme Gradient Boosting, is a scalable, distributed gradient-boosted decision tree (GBDT) machine learning library. It provides parallel tree boosting and is the leading machine learning library for regression, classification, and ranking problems.
